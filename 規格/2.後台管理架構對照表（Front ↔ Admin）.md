# 2. 後台管理架構對照表（Front ↔ Admin）

後台管理架構對照表（Front ↔ Admin）

後台管理系統以實務操作為導向，僅提供電腦版介面，著重功能清楚與操作效率，不以視覺設計或多裝置版型為目標。
1) 首頁 Home（前台呈現內容）
前台內容
●	精選/熱門課程

●	學員見證摘要

●	最新文章/公告摘要

●	首頁主視覺與 CTA

後台對照管理
●	✅ 精選/熱門課程設定（指定課程、排序）

●	✅ 學員見證摘要


________________________________________
2) 課程 Courses（列表/詳情/播放）
前台內容
●	課程列表、分類、搜尋（可選）

●	課程詳情（簡介、大綱、試看）

●	播放頁（Mux 播放、章節、續播、權限判斷）

後台對照管理
●	✅ 課程管理

○	新增/編輯/刪除 /上下架

○	課程封面圖、標題、文字簡介、標籤/分類

○	課程排序（編號序）

○	試聽與否

●	✅ 章節/單元管理

○	新增/編輯/排序

○	綁定影片（Mux Playback ID / Asset）

●	✅ 權限規則設定（簡化版）

○	此課程需要「訂閱」或「單次購買」(二選一或多選)

●	✅ 播放頁資料支援

○	章節列表輸出

○	續播資料（觀看進度）讀取/寫入

________________________________________
